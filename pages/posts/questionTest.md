```
/*21级的同学实在是太卷辣
驴哒同学为了给自己喜欢的虚拟主播打广告，励志成为刷题榜榜一，于是他每天激励自己刷题
已知驴哒同学最初每小时能刷两道任意难度的题，如果 当天 心情好可以使 当天 的刷题效率提升1道/小时（即每小时2+1道题），同时人的效率也会受到天气的影响，若天气是晴天则 当天 的刷题效率多提升1道/小时
同时驴哒同学对自己严格要求，如果 当天 刷题时间小于等于2小时，则 第二天 的刷题效率提升2道/小时；同时如果刷题时间大于等于10小时，则会因为 第二天 过于劳累使刷题效率降低1道/小时
刷题效率为可继承，第二天不会从2开始。
试计算X天后驴哒同学一共刷了多少道题？

输入
第一行输入一个整数 X ，为在 X 天内驴哒同学一共刷了多少道题
后面X行输入整数 t，f，w，分别代表驴哒同学当天的刷题时间，心情和当天的天气
(可以用0和1来代表天气晴雨和心情好坏)

输出
输出一个整数，代表X天内驴哒同学一共刷的题

题数n
心情f
刷题时长t
天气w
m行
w=1 n++
t>10 n--
t《2 n++++
f=1 n++

样例
7
5 1 1
11 0 1
3 1 0
1 1 1
9 0 0
4 1 0
6 1 1

290 

*/
#include <stdio.h>

int main()
{
    int i,n=0,f,t,w,m,p=2,x;
    scanf("%d",&x);
    for(i=0;i<x;i++){
        scanf("%d %d %d",&t,&f,&w);
        if(w==1){
            p++;
        }
        if(f==1){
            p++;
        }
        n = n + p*t;
        if(t>=10){
            p--;
        }else if(t<=2){
            p=p+2;
        }
    }
    printf("%d",n);

    return 0;
}
```
